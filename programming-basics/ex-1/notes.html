Exercise 1: Setting Up npm

Level 1: Basic npm Setup

I initialized a new Node.js project using npm init, added a start script to run a basic index.js file, and added a dev script to run the file with debugging output using DEBUG=* node index.js.

  • Initialized a new Node.js project using npm init.
  • Added a start script to run index.js.
  • Added a dev script to show debugging output.
npm init -y
npm start
npm run dev

Level 2: Using Chalk and Nodemon

I installed the chalk package and used it in index.js to style console outputs. I also added nodemon as a dev dependency and used it to automatically restart the program during development.

  • Styled console messages using chalk in index.js.
  • Installed nodemon and used it to restart the program automatically during development.
npm install chalk
npm install --save-dev nodemon

Level 3: Prestart Script

I added a prestart script to ensure node_modules is available before running the program. I tested this by removing node_modules and ensuring that npm start runs successfully after reinstalling dependencies.

  • Added a prestart script to check for node_modules before running the program.
  • Verified that npm start works after removing and reinstalling dependencies.
"scripts": { "start": "node index.js", "dev": "DEBUG=* node index.js", "prestart": "npm install" }
