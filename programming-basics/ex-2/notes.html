Exercise 2: Command-Line Program

Level 1: Basic Sum Calculation

I created a script that calculates the sum of numbers passed as command-line arguments. I used process.argv to parse and handle the arguments. The script also includes error handling for invalid inputs, such as non-numeric values.

  • Created a script to calculate the sum of command-line arguments.
  • Used process.argv to parse and handle the arguments.
  • Implemented error handling for invalid inputs (e.g., non-numeric values).
node sum.js 1 2 3

Level 2: Multiply Flag

I enhanced the script to accept a --multiply flag to multiply the numbers instead of adding them. I also updated the help text to include usage instructions for both sum and multiply operations.

  • Added a --multiply flag to perform multiplication.
  • Updated the help text to include usage instructions.

Level 3: Modularization and Global Command

I modularized the program by creating a separate file (mathutils.js) for the add and multiply functions. The program now accepts an operation followed by operands as arguments, like command add 1 2 or command multiply 3 4. I also created a script using the bin part of the package.json and used npm link to test the script as a global command.

  • Created a mathutils.js file for the add and multiply functions.
  • Modified the script to accept operation and operands as arguments.
  • Created a global command using npm link.
