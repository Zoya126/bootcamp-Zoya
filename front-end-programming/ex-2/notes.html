1. Initialize a New Project Using a Package Manager

  • Start by initializing a new npm project using: npm init -y
  • Install TypeScript and other necessary packages: npm install typescript --save-dev
  • Create a tsconfig.json file for TypeScript configuration: npx tsc --init

2. Configure TypeScript Using a tsconfig.json File

  • Modify the tsconfig.json to include the necessary options for your project, such as setting the target, module, and outDir.
  • Example configuration: 
    {
    "compilerOptions": {
        "target": "es6",
        "module": "commonjs",
        "outDir": "./dist",
        "strict": true,
        "esModuleInterop": true
    },
    "include": ["src/**/*.ts"],
    "exclude": ["node_modules"]
}

3. Problems to Solve

Problem 1: Load a JSON file, render a table dynamically, and include sorting functionality

  • Use TypeScript to load the JSON file and dynamically render a table.
  • Implement sorting functionality for the table columns.

Problem 2: Fetch Data from a CSV File Using PapaParse

  • Use the PapaParse library to fetch and parse CSV data.
  • Enhance the table with column filtering and row highlighting features.

Problem 3: Integrate a Build Tool (e.g., tsc for Transpilation)

  • Set up the tsc tool for transpilation from TypeScript to JavaScript.
  • Ensure the app works in both development and production environments.

4. Dev, Test, and Production Life Cycle

  • Use vite in your package.json for:
    • Development server with live reload.
    • Linting and type-checking during development.
    • Production build with bundling.
