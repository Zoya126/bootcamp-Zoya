Learn CSS Variables

Level 1: Basic Use of CSS Variables

  • Define CSS Variables: In this level, you define basic CSS variables in a separate file (e.g., variables.css) for easy management of reusable values across your stylesheet.
    • Variables can include:
      • Colors (e.g., --primary-color, --secondary-color)
      • Font sizes (e.g., --header-font-size, --body-font-size)
      • Spacing (e.g., --margin-spacing)
  • Link Variables to Main CSS: You link your variables.css to the main stylesheet using @import, allowing you to use these variables throughout your site.
  • Apply Variables: Use the defined variables in your CSS to style elements (e.g., applying colors, font sizes, margins).

Level 2: Advanced Use of CSS Variables

  • Default Values for Variables: When defining CSS variables, you can provide a fallback value using var(--variable, fallback). If the variable is not set, the fallback will be used.
    • Example: color: var(--primary-color, black);
  • Targeting Specific Elements: Use CSS selectors like nth-child to style specific elements. For example, alternate rows of a table can be styled differently. This helps with more complex styling, like creating striped tables.

Level 3: Dynamic Theming and Nested Variables

  • Using Variables to Define Other Variables: You can use one variable to define another, making your CSS even more flexible.
    • Example: --secondary-color: var(--primary-color);
  • Dynamic Theme Switching: This level introduces the concept of creating multiple themes (like light and dark modes) using CSS variables. By toggling classes (e.g., .dark-mode), you can easily change the entire theme by updating variables for colors, background, text, etc.
    • This can be achieved using JavaScript to toggle classes or by using media queries to automatically detect the user's preference (e.g., prefers-color-scheme).
