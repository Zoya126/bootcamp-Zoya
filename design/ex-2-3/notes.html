What I Learned from the Basics of Icons Exercise

Level 1: Basic Icon Usage

  • Learned how to add icons to a webpage using an icon font library (e.g., Font Awesome).
  • Styled icons using CSS to adjust their size, color, and hover effects.
  • Gained familiarity with how to use the `` tag for font icons and customize them.

Level 2: SVG Icons and Grid Layout

  • Replaced one icon with an inline SVG to demonstrate how to use SVGs for icons.
  • Created a grid layout of six icons (three from Font Awesome and three as inline SVGs) using CSS Grid.
  • Added hover effects to both Font Awesome and SVG icons, such as color change and scaling on hover.

Level 3: Navigation Bar and Buttons

  • Designed a navigation bar with interactive icons (home, about, and contact) using Font Awesome icons.
  • Aligned icons with labels in the navigation bar using Flexbox for better layout control.
  • Created download and upload buttons using SVG icons, demonstrating how to embed SVGs in buttons.
  • Learned how to apply consistent spacing and alignment to icons to create a clean, user-friendly layout.
