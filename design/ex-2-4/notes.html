Summary Notes: Basics of Images

Introduction

  • Image Types:
    • JPEG: Best for photos due to lossy compression and small file size.
    • PNG: Supports transparency and lossless compression, suitable for graphics and logos.
    • SVG: Scalable vector format, ideal for icons and illustrations.
  • Basic Image Editing:
    • Adjusting transparency, resizing, and compressing images.
  • Advanced Techniques:
    • Full-bleed images for headers or backgrounds.
    • Using placeholders like Lorem Picsum for mock images.
    • Implementing responsive images for different screen sizes.

What I Learned

  • How to add and style images using CSS properties like width, height, and border-radius.
  • Editing images for web optimization, including size adjustment and compression.
  • Adding overlays like gradients or tints using CSS for better visual appeal.
  • Using the <picture> element to create responsive images with high-res and low-res versions.
  • Optimizing image loading with lazy loading techniques for improved performance.
